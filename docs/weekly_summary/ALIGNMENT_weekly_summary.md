# 对齐文档：AI 周报总结

## 1. 项目背景
FlashOfThought 是一个语音驱动的想法记录和管理工具。目前支持：
- 录制语音/文本想法。
- AI 处理以结构化想法（标题、摘要、标签、核心观点等）。
- 语义搜索和与知识库聊天。
- 知识图谱可视化。
- 基本的笔记增删改查操作。

## 2. 需求理解
**目标**：实现“AI 周报总结”功能，自动聚合和总结过去一周记录的想法。

**输入**：
- 过去 7 天内创建的所有笔记。

**输出**：
- 包含以下内容的结构化摘要：
    - **总数**：记录的想法数量。
    - **主题细分**：想法的分类（例如，“AI 产品 (5)”，“创业想法 (3)”）。
    - **关键亮点**：3-5 个最重要或有趣的想法，并附带简要描述。
    - **趋势分析**：（可选）关于用户本周关注点的简要评论。

**用户界面**：
- “知识回顾”中一个新的按钮或部分（或单独的标签页）来触发生成。
- 以清晰易读的格式（Markdown/卡片）显示结果。

## 3. 歧义与决策
- **触发方式**：用户手动点击“生成周报”（“自动生成”通常意味着“按需”或“定时”，但对于本地应用，按需更好）。
    - **决策**：在“知识回顾”标签页中添加“📅 生成本周周报”按钮。
- **日期范围**：“本周”通常意味着周一到周日或“过去 7 天”。
    - **决策**：使用“过去 7 天”（滚动窗口），因为它对个人工具更灵活，或者提供选择器。默认为“过去 7 天”。
- **存储**：摘要应该被保存吗？
    - **决策**：不，目前是即时生成的。如果用户想保存，可以复制它。（MVP 范围）。

## 4. 验收标准
- [ ] 用户可以点击按钮生成过去 7 天的摘要。
- [ ] 系统正确过滤过去 7 天的笔记。
- [ ] LLM 生成指定格式的摘要（计数、主题、亮点）。
- [ ] UI 正确显示摘要。
- [ ] 如果没有找到笔记，显示友好的提示信息。
