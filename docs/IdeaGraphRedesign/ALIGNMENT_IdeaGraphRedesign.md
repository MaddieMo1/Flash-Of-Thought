# ALIGNMENT_IdeaGraphRedesign

## 1. 原始需求
用户要求重新设计知识图谱的界面、功能及交互方式。
要求：
- 简约、交互友好。
- 参考提供的图片（深色背景、霓虹光晕、环形布局、力导向布局）。
- 替换原有的 3D 星系视图，改用更直观的 2D 视图。

## 2. 核心变更点
- **视觉风格**: 深色模式 (#0E1117)，高亮节点 (#00F0FF, #FF0055 等)，发光效果。
- **布局算法**: 支持 **力导向图 (Force)** 和 **环形布局 (Circular)** 切换。
- **交互方式**: 
  - 支持滚轮缩放、拖拽平移 (Roam)。
  - 点击/悬停高亮相邻节点 (Focus Adjacency)。
  - 显示/隐藏标签。
- **技术选型**: 从 `plotly.graph_objects` (3D) 迁移到 `streamlit-echarts` (2D Canvas/SVG)，以获得更好的性能和交互体验。

## 3. 验收标准
- [x] 成功安装并集成 `streamlit-echarts`。
- [x] 界面能够正确渲染后端 `/graph` 数据。
- [x] 支持布局切换（Force/Circular）。
- [x] 节点颜色根据分类自动映射（技术、应用、产品等）。
- [x] 交互流畅，无明显卡顿。
